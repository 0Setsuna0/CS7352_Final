# Results Directory

This directory stores reproducible benchmark and quality outputs.

Expected files:

- `benchmark.csv`: rows written by `scripts/run_cogvideox_accel.py`
- `quality_metrics.csv`: rows written by `scripts/evaluate_quality.py`
- `summary.md`: human-readable experiment summary
- `videos/`: generated videos
- `logs/`: per-run metadata JSON files

Do not fill in numbers by hand unless they came from an actual run.
